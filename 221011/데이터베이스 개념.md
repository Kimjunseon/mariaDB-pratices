## DB(DataBase) 기본
 - 데이터의 집합
 - 여러 응용 시스템들의 [통합]된 정보들을 [저장]하여 [운영]할 수 있는 [공용] 데이터의 집합
 - 저장, 검색, 갱신, 연관, 조직화
 - 오라클, mysql 등..
 - DA(Data Architect)
 - DBA(Data Administrator)
 - DB Developer

## DB 특성
 - 실시간 접근성
 - 계속적인 변화
 - 동시 공유성
 - 내용 참조

## DBMS
 - DDL 정의기능 (학생, 성적 등..) # create table, drop table..
 - DCL 조작기능 DB에 접근할 수 있는 명령어들
 - DML 제어기능 (CRUD 구문)
   - select
     - 기본
     - 통계
     - 조인
     - 서브쿼리
   - insert
   - update
   - delete
## DB 종류
 - 객체 관계형 DB
 - NoSQL(대용량 데이터의 웹 서비스와 SNS, 클라우드 컴퓨팅의 대중화로 주목받는 DB 기술)
